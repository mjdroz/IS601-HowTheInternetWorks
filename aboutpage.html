About Page

This website is an assignment for a graduate level course, IS 601 - Web Systems Development. This course is offered at New Jersey Institute of Technology in Newark New Jersey.

This assignment titled, "How the Internet Works", is an information based assignment, which is going to help students gain a better understanding of the internet. The main goal behind this is, if you are going to work with the internet or development products on it, you better know how it works in the first place!

The main internet terms and organizations that I will define and give examples of in this assignment include:
1. Domain names and ICANN
2. IP addresses, Packets and Routing
3. DNS
4. HTTP and HTTPS protocols
5. W3C and HTML and CSS

Finally, any external sources will be linked at the bottom of each page for your information/research.
